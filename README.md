# 🍽️ Restaurant Data Analysis

## 📌 Overview

This project analyzes restaurant data to uncover insights related to cuisines, pricing, ratings, and delivery trends.

---

## 📊 Key Insights

* Popular cuisines dominate customer preferences
* Metro cities have higher restaurant density
* Mid-range pricing is most common
* Online delivery improves accessibility
* Restaurants cluster in urban regions
